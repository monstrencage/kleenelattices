module Lts: sig .. end
Direct constructions of LTS from expressions, and LTS simulation.
val trad : string Expr.expr -> Tools.lts
Computes an LTS from an identity-free expression.
val simul : Tools.lts -> Tools.lts -> string Expr.ground option
Tries to find a ground term of the first LTS that is not recognised by the second LTS. In case of success, returns said term. Otherwise returns None.
